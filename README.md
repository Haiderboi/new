# Acme Store (Next.js 15)

Minimal e-commerce storefront with Electronics and Fashion categories, Supabase product loading, and a slide-out WhatsApp checkout cart.

## Environment Variables

Create `.env.local` with:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url_here
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key_here
# Optional (defaults to 923000000000)
NEXT_PUBLIC_WHATSAPP_NUMBER=923000000000
```

## Run

```bash
npm install
npm run dev
```
